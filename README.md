# Smart Civic Issue Reporting System

A web-based solution for instant reporting of civic issues.

## Introduction

The Smart Civic Issue Reporting System is a web-based platform designed to empower citizens to report civic issues directly to the concerned authorities. This system aims to bridge the gap between citizens and municipal corporations, ensuring timely resolution of problems.

## Key Features

*   **Image-based Reporting:** Users can upload images of the issue for better clarity.
*   **Real-time Tracking:** A unique complaint ID is generated for each report to track its status.
*   **Automated Complaint Routing:** The system is designed to automatically route complaints to the relevant department (future scope).
*   **Geolocation:** Users can share their current location for accurate issue mapping.

## Tech Stack

*   **Frontend:** HTML, CSS, JavaScript
*   **Backend:** Node.js, Express.js
*   **Database:** A simple JSON file is used as a database for this prototype.

## How to run the project

1.  Clone the repository.
2.  Run `npm install` to install the dependencies.
3.  Run `node server.js` to start the server.
4.  Open `index.html` in your browser.
